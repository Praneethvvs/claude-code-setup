# Project: My Claude Setup

## Overview
This project contains configuration, scripts, and utilities for setting up and customizing Claude Code workflows.

## Conventions
- Keep configuration files well-documented with inline comments
- Use descriptive names for scripts and utilities
- Test changes locally before committing

## Project Structure
- `.claude/` - Claude Code project configuration
- `scripts/` - Automation and setup scripts
- `configs/` - Reusable configuration templates

## Workflow Preferences
- Prefer simple, readable solutions over clever abstractions
- Always read existing files before modifying them
- Use git for version control; commit with clear messages
